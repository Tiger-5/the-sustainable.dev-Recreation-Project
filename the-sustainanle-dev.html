What is the sustainable.dev?

The sustainable.dev is a series of developer resources, blog posts and long-form case studies that help to inform the web design and development community of the need for change and to encourage action to make the web greener and more sustainable.


With more knowledge on which tools, web hosts and coding practices are more sustainable, the aim is for developers to use this knowledge to improve on the current state of the web.
